# 因子扩展 — 股票域（Universe）字段 + 自定义域文件上传

## 背景

因子的绩效指标跟特定股票域绑定，同一个因子在不同域上表现不同。新建 Py 文件因子时，必须声明回测的股票域。

股票域有三个维度：
1. **标准指数** — 沪深300、中证500、中证1000、上证50、中证2000、全市场等（已有预设列表）
2. **申万行业** — 食品饮料、银行、电子等（已有预设列表）
3. **自定义** — 用户上传股票列表文件 + 必须填写域形成逻辑

## 数据库改动

在 `factor_metadata` 表新增以下字段：

```sql
ALTER TABLE factor_metadata ADD COLUMN universe_type VARCHAR(20) DEFAULT '';
ALTER TABLE factor_metadata ADD COLUMN universe_id VARCHAR(50) DEFAULT '';
ALTER TABLE factor_metadata ADD COLUMN universe_desc VARCHAR(500) DEFAULT '';
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `universe_type` | varchar(20) | 域类型：`index` / `industry` / `custom` |
| `universe_id` | varchar(50) | 预设域ID（如 `hs300`、`zz500`、`sw_food`），自定义时为 `custom` |
| `universe_desc` | varchar(500) | 自定义域的形成逻辑描述（仅 custom 类型必填） |

## PUT 接口扩展

在 `PUT /api/v1/factor/my/:id` 的可更新字段白名单中，新增：

- `universe_type`
- `universe_id`
- `universe_desc`

## 自定义股票列表文件上传

复用已有的 source 文件上传模式，新增一对接口：

### 上传自定义股票列表

```
POST /api/v1/factor/my/:id/universe-file
Content-Type: multipart/form-data
X-API-Key: ak_xxx
```

**请求参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | file | 是 | `.csv` 或 `.txt` 文件 |

**MinIO 存储**：

```
bucket: factor-sources
key: {factor_id}_universe.csv
```

**成功响应**：

```json
{
  "success": true,
  "message": "Universe file uploaded successfully",
  "data": {
    "file_name": "my_stocks.csv",
    "file_size": 2048,
    "uploaded_at": "2026-05-26T04:00:00Z"
  }
}
```

**约束**：
- 接受 `.csv` 和 `.txt` 后缀
- 文件大小限制 1MB
- 重复上传覆盖旧文件

### 获取自定义股票列表

```
GET /api/v1/factor/my/:id/universe-file
X-API-Key: ak_xxx
```

**成功响应**：

```json
{
  "success": true,
  "data": {
    "file_name": "yuyang_372_universe.csv",
    "file_size": 2048,
    "content": "SH600000\nSH600036\nSZ000001\n...",
    "uploaded_at": "2026-05-26T04:00:00Z"
  }
}
```

### 删除自定义股票列表

```
DELETE /api/v1/factor/my/:id/universe-file
X-API-Key: ak_xxx
```

删除因子时自动清理（与源码文件同理）。

## 请求示例

```bash
# 更新股票域信息（预设）
curl -X PUT http://61.151.241.233:8080/api/v1/factor/my/yuyang_372 \
  -H "X-API-Key: ak_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "universe_type": "index",
    "universe_id": "hs300"
  }'

# 更新股票域信息（自定义）
curl -X PUT http://61.151.241.233:8080/api/v1/factor/my/yuyang_372 \
  -H "X-API-Key: ak_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "universe_type": "custom",
    "universe_id": "custom",
    "universe_desc": "日均成交额>5000万 & 总市值>50亿 & 非ST & 上市满1年"
  }'

# 上传自定义股票列表
curl -X POST http://61.151.241.233:8080/api/v1/factor/my/yuyang_372/universe-file \
  -H "X-API-Key: ak_xxx" \
  -F "file=@/path/to/stock_list.csv"

# 获取自定义股票列表
curl http://61.151.241.233:8080/api/v1/factor/my/yuyang_372/universe-file \
  -H "X-API-Key: ak_xxx"
```
