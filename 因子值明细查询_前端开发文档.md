# 因子值明细查询 - 前端开发文档

## 背景

回测引擎新增了因子值明细输出功能。每次回测完成后，会将每只股票每天的因子值写入 `backtest_factor_values` 表（含约 60 天预热期数据）。

**数据量参考：**
- 1000 只股票 × 500 天 ≈ 50 万条
- 5000 只全市场 × 500 天 ≈ 250 万条

---

## API 接口

**Base URL：** `/api/v1/backtest`

**鉴权方式：** 仅需 API Key 认证（`AuthenticateOnly`），不需要配置接口权限。与 `stock-pools`、`price-type-options` 同级。

---

### 1. 因子值分页查询

```
GET /api/v1/backtest/task/:id/factor-values
```

**场景：** 查看某天所有股票的因子值排名表格，或某只股票的因子值时序。

**请求参数：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| trade_date | string | 否 | - | 筛选日期，如 `2025-06-15` |
| stock_code | string | 否 | - | 筛选股票，如 `SH.600519` |
| sort_by | string | 否 | `factor_value` | 排序字段：`factor_value` / `stock_code` / `trade_date` |
| sort_order | string | 否 | `desc` | 排序方向：`asc` / `desc` |
| page | int | 否 | 1 | 页码 |
| page_size | int | 否 | 50 | 每页条数（最大 500） |

**响应：**

```json
{
  "success": true,
  "data": {
    "task_id": "BT_20260126_abc12345",
    "records": [
      {
        "trade_date": "2025-06-15",
        "stock_code": "SH.600519",
        "factor_value": 3.1415
      },
      {
        "trade_date": "2025-06-15",
        "stock_code": "SZ.000858",
        "factor_value": 2.7182
      }
    ],
    "total": 5000,
    "page": 1,
    "page_size": 50
  }
}
```

**使用示例：**

```
# 查看 2025-06-15 的截面排名（因子值从大到小）
GET .../factor-values?trade_date=2025-06-15&sort_by=factor_value&sort_order=desc

# 查看某只股票的因子值走势
GET .../factor-values?stock_code=SH.600519&sort_by=trade_date&sort_order=asc&page_size=500

# 查看某天因子值最小的 10 只
GET .../factor-values?trade_date=2025-06-15&sort_by=factor_value&sort_order=asc&page_size=10
```

### 2. 可用交易日列表

```
GET /api/v1/backtest/task/:id/factor-values/dates
```

**场景：** 日期选择器、日期滑块。无需请求参数。

**响应：**

```json
{
  "success": true,
  "data": {
    "task_id": "BT_20260126_abc12345",
    "dates": [
      "2024-10-08",
      "2024-10-09",
      "2024-10-10",
      "...",
      "2025-12-31"
    ],
    "total": 560,
    "backtest_start_date": "2025-01-02"
  }
}
```

> **`backtest_start_date`** 标识回测正式开始的日期。该日期之前的是预热期数据（因子值有效，但没有对应的 IC 等回测指标）。前端可在日期滑块上做分界标记。

### 3. 截面统计

```
GET /api/v1/backtest/task/:id/factor-values/stats
```

**场景：** 展示某天的因子值统计摘要。

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| trade_date | string | 是 | 日期，如 `2025-06-15` |

**响应：**

```json
{
  "success": true,
  "data": {
    "trade_date": "2025-06-15",
    "count": 4850,
    "mean": 0.0523,
    "std": 1.2345,
    "min": -4.567,
    "max": 5.678,
    "percentiles": {
      "p5": -1.89,
      "p25": -0.65,
      "p50": 0.04,
      "p75": 0.72,
      "p95": 2.11
    }
  }
}
```

### 4. 因子值分布（直方图数据）

```
GET /api/v1/backtest/task/:id/factor-values/distribution
```

**场景：** 因子值分布直方图，服务端已做分箱计算，前端直接渲染。

**请求参数：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| trade_date | string | 是 | - | 日期 |
| bins | int | 否 | 50 | 分箱数量（5 ~ 200） |

**响应：**

```json
{
  "success": true,
  "data": {
    "trade_date": "2025-06-15",
    "bins": 50,
    "histogram": [
      { "range_start": -4.57, "range_end": -4.37, "count": 3 },
      { "range_start": -4.37, "range_end": -4.16, "count": 8 },
      { "range_start": -4.16, "range_end": -3.96, "count": 15 }
    ],
    "total": 4850
  }
}
```

> 当所有因子值相同时，返回 `bins=1` 的单条记录。

---

### 5. 因子值下载（CSV）

```
GET /api/v1/backtest/task/:id/factor-values/download
```

**场景：** 用户导出因子值数据做进一步研究。

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| trade_date | string | 否 | 筛选日期（不传则导出全部） |

**响应：** CSV 文件流（GBK 编码，Excel 可直接打开）

```
Content-Type: text/csv; charset=gbk
Content-Disposition: attachment; filename="factor_values_BT_xxx_2025-06-15.csv"
```

CSV 内容：

| 交易日期 | 股票代码 | 因子值 |
|----------|----------|--------|
| 2025-06-15 | SH.600000 | 1.234567 |
| 2025-06-15 | SH.600004 | 0.876543 |

> 因子值保留 6 位小数。按日期升序、因子值降序排列。

## 前端页面建议

### 在回测结果页新增「因子值」Tab

现有的回测结果页已有「概览」「分层收益」「IC 序列」等 Tab，建议新增一个 **「因子值明细」** Tab，包含以下组件：

#### 组件 1：日期选择器

- 调用 `GET .../factor-values/dates` 获取日期列表
- 默认选中回测最后一天
- 可用滑块或下拉框切换日期
- 用 `backtest_start_date` 在滑块上标记预热期分界线

#### 组件 2：截面统计卡片

- 调用 `GET .../factor-values/stats?trade_date=xxx`
- 展示选中日期的统计摘要：股票数、均值、标准差、最大/最小、中位数
- 切换日期时自动刷新

#### 组件 3：因子值分布图

- 调用 `GET .../factor-values/distribution?trade_date=xxx`
- 展示直方图（推荐 ECharts bar 图）
- X 轴：因子值区间，Y 轴：股票数量
- 可叠加正态分布曲线参考

#### 组件 4：截面排名表格

- 调用 `GET .../factor-values?trade_date=xxx&sort_by=factor_value&sort_order=desc`
- 展示表格：排名 | 股票代码 | 因子值
- 支持分页、搜索股票代码
- 点击股票弹出该股票的因子值时序走势

#### 组件 5：因子值导出按钮

- 调用 `GET .../factor-values/download?trade_date=xxx`
- 支持导出当天截面数据或全量数据

## 交互流程

```
用户打开回测结果页
    │
    ├─ Tab: 概览 / 分层收益 / IC 序列 / 【因子值明细】
    │
    └─ 点击「因子值明细」Tab
        │
        ├─ 1. GET .../factor-values/dates → 渲染日期选择器
        │     默认选最后一天
        │
        ├─ 2. GET .../factor-values/stats?trade_date=xxx → 渲染统计卡片
        │
        ├─ 3. GET .../factor-values/distribution?trade_date=xxx → 渲染直方图
        │
        └─ 4. GET .../factor-values?trade_date=xxx → 渲染排名表格
              │
              ├─ 切换日期 → 重新请求 2、3、4
              │
              ├─ 点击某只股票 → GET .../factor-values?stock_code=xxx
              │   → 弹窗显示时序走势图
              │
              └─ 点击导出 → GET .../factor-values/download
```

---

## 注意事项

1. **数据量大，必须分页**：全市场回测某天的截面数据可达 5000 条，全量因子值可达百万级
2. **预热期数据**：`backtest_start_date` 之前的日期是因子预热期数据，因子值有效但没有对应的回测指标（IC 等）
3. **NaN 值已过滤**：回测引擎在写入时已跳过 NaN 和空值，前端不需要额外处理
4. **CSV 编码**：下载的 CSV 是 GBK 编码，Excel 可直接打开不乱码
5. **与现有 daily-metrics 的关系**：
   - `daily-metrics`：每天一条汇总记录（IC、分层收益等）
   - `factor-values`：每天 × 每只股票的明细记录
   - 两者通过 `task_id + trade_date` 关联
