# v1.7.8 更新日志

发布日期：2024-11-29

## 🚀 新功能

### 1. SSH 远程文件夹关联
- 支持通过 SSH 连接远程服务器上的项目文件夹
- 自动检测 Linux/Windows 服务器类型
- 可视化文件夹浏览器，双击进入目录
- 支持远程 Git 仓库初始化和配置

### 2. 智能 .gitignore 配置
- 关联项目时自动检测并提示配置 .gitignore
- 提供常见文件类型的可视化勾选（日志文件、临时文件、依赖包等）
- 支持本地和 SSH 远程项目

### 3. 强制版本标签
- 提交代码时必须创建版本标签
- 自动显示最新版本号，引导用户填写下一版本
- 标签格式统一为小写 `v` 前缀（如 v1.7.8）
- 标签查重支持大小写不敏感

## 🔧 优化

### UI/UX 改进
- 「本地关联」更名为「关联项目文件夹」
- 「本地变更」更名为「本地 Git 仓库变更」
- 按钮文案优化：「比对 / 提交到本地 Git 仓库」
- 新增「打开 .git 文件夹」按钮（IT 用户调试用）

### 仓库管理重构
- 原「本地管理」拆分为「仓库配置」和「代码同步」两个区域
- 关联信息下方显示 .gitignore 配置状态

## 🐛 修复

- 修复换行符导致的虚假文件变更问题（新增 .gitattributes）
- 修复版本标签显示错误（v/V 大小写混乱）
- 修复 SSH 文件浏览器样式问题

## 📦 构建

- 新增 Gitea Actions 工作流配置
- 优化 TypeScript 类型定义
