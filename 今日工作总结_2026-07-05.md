# 今日工作总结（2026-07-05）

## 一、完成事项

### 1. 回测任务列表 - 轮询刷新闪烁问题修复
- **文件**：`src/renderer/views/FactorLibrary/Backtest/TasksContent.vue`
- **问题**：每 5 秒自动轮询刷新时，整屏被 loading 遮罩盖住，视觉体验差
- **方案**：给 `loadTasks` 加 `silent` 参数，轮询调用时（`silent=true`）不设置 `loading.value = true`；手动点刷新 / 翻页 / 筛选仍走原逻辑显示遮罩
- **commit**：`c3333ba`

### 2. 新建因子数据源 - 按 engine+database 精确路由（前端配套改造）
- **文件**：`src/renderer/views/FactorLibrary/MyFactors.vue`
- **背景**：后端数据中心改造为多引擎多库模型，要求前端精确传递 `(engine, database, table)` 三件套
- **改动**（共 6 处）：
  1. `DataSourceItem` 接口新增 `engine` 字段
  2. `handleAddDataSource` 初始化默认值
  3. `selectSearchResult` 从搜索结果回填 engine + database
  4. `handleTableChange` 修正 `getTableFields(engine, database, table)` 传参（核心）
  5. `clearSelectedTable` 重置时同步重置 engine
  6. UI 标签显示 `engine / database`
- **commit**：`b859cc5`
