# @openim/shared

Shared utilities, hooks, components, and types used across OpenIM packages.

## Installation

```bash
pnpm add @openim/shared
```

## Exports

- `cn`
- `useDraggableFloating`
- `useWindowSize`
- `useElementRect`, `useBodyRect`, `useRefRect`
- `useComposedRef`
- `useUnmount`
- `useThrottledCallback`
- `useIsBreakpoint`
- `useScrolling`
